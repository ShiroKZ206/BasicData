Post 1: Apa yang Anda harapkan dari MK Basis Data

Saya berharap dengan mengikuti mata kuliah Basis Data bisa memahami cara menyimpan dan mengelola data dengan benar. Selain itu, saya ingin mampu merancang database yang baik dan bisa diterapkan pada aplikasi nyata.

Post 2: Apa itu Basis Data

Basis data adalah kumpulan data yang terstruktur dan saling berhubungan, disimpan dalam sistem agar mudah diakses dan dikelola. Contoh penerapannya ada di perbankan, e-commerce, maupun sistem akademik.

Post 3: Apa itu Conceptual DB

Conceptual Database adalah rancangan database pada level konseptual yang menggambarkan entitas, atribut, dan hubungan antar entitas. Biasanya divisualisasikan menggunakan diagram ERD.

Post 4: Apa itu Physical Model

Physical Model adalah rancangan database pada level fisik, yaitu bagaimana data benar-benar disimpan di DBMS. Di tahap ini dibuat tabel, tipe data, primary key, foreign key, dan indeks.
